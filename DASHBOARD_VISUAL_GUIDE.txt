COLOR CODING LEGEND
================================================================================

CLASSIFICATION COLORS:

  🟢 GREEN
    │ Safe, reversible actions
    │ Examples: cat, ls, SELECT
    │ Decision: Auto-approve
    │ Color: #10B981 (bright green)
    │
    ├─ Badge: [APPROVED] (green background)
    └─ Confidence: Usually 90%+

  🟡 YELLOW
    │ Risky but reversible actions
    │ Examples: CREATE TABLE, pip install
    │ Decision: Flag and proceed
    │ Color: #F59E0B (bright yellow)
    │
    ├─ Badge: [FLAGGED] (yellow background)
    └─ Confidence: Usually 60-80%

  🔴 RED
    │ Dangerous, destructive actions
    │ Examples: rm -rf /prod, DROP TABLE
    │ Decision: Block immediately
    │ Color: #EF4444 (bright red)
    │
    ├─ Badge: [BLOCKED] (red background)
    └─ Confidence: Usually 90%+
